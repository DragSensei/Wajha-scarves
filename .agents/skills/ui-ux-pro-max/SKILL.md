# UI-UX Pro Max Skill: Premium Frontend Engineering

This skill enables you to design and implement premium, modern user interfaces that wow users instantly.

## Core Aesthetic Standards

1. **Curated Color Palettes**:
   - Avoid generic, basic colors (like `bg-red-500` or `text-blue-600`).
   - Use sophisticated HSL-based palettes with cohesive neutrals (slate, zinc, stone) and vibrant, tailored accents (indigo, violet, emerald, amber).
   - Ensure excellent contrast for readability (WCAG AA compliant).

2. **Typography**:
   - Always load premium modern sans-serif fonts (e.g., Inter, Outfit, Roboto, Outfit) instead of system defaults.
   - Establish clear type scale hierarchy with distinct headings and body styles.

3. **Glassmorphism & Backdrops**:
   - For overlays, modals, and headers, use translucent background blur:
     `bg-white/70 backdrop-blur-md border border-white/20` (or matching dark mode equivalent).
   - Use soft shadow layers for depth: `shadow-sm`, `shadow-md`, `shadow-xl`.

4. **Micro-Animations & Transitions**:
   - All interactive elements (buttons, links, cards) must have smooth transitions: `transition-all duration-300 ease-in-out`.
   - Implement hover micro-animations (e.g., scale-up by 1% to 2%, subtle translation offsets, opacity changes).

5. **Responsive & Mobile First**:
   - Layouts must be responsive and adapt gracefully to all screen sizes (mobile, tablet, desktop).
   - Use CSS Grid and Flexbox for flexible structure.

6. **Interactive Visual Cues**:
   - Active states, loading states, empty states, and errors must look polished and intentional.
   - Use high-quality icons (SVG icons or icon libraries) consistently.
